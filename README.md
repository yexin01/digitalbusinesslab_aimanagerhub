# AI Manager Hub

This is a [Next.js](https://nextjs.org/) project designed to provide an AI-powered dashboard for managers to overview employee information and performance.

## Live Demo

The application is deployed on Vercel and can be accessed here:
[https://aimanagerhub.vercel.app/](https://aimanagerhub.vercel.app/)

## Getting Started Locally

To run the project on your local machine, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yexin01/digitalbusinesslab_aimanagerhub.git
    cd digitalbusinesslab_aimanagerhub
    ```

2.  **Install dependencies:**
    Make sure you have Node.js (v18 or newer recommended) and npm installed.
    ```bash
    npm install
    ```

3.  **Run the development server:**
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

The page auto-updates as you edit files in the `src` directory.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load fonts like Inter and Roboto Mono.
